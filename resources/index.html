Repositories

Here you can find multiple open repositories that may help you to develop smart microscopes in your lab!

Name Language Status Description
Microscope Automation Python Inactive, stable Microscope Automation is an automation software for a variety of popular microscopes, such as Zeiss and 3i.
ImSwitch Python Active, stable ImSwitch is a software solution in Python that aims at generalizing microscope control for flexible control of multiple microscope modalities.
Adaptive-imaging Python Inactive, stable Adaptive Imaging is a set of tools for adaptive imaging. The overall idea of this project is to adapt the imaging speed of a microscope to the presence of events in the images.
Event-driven acquisition (EDA) Python Active, stable event-driven-acquisition is a Python library for advanced microscope control routines to enable acquisitions to react to specific biological events of interest.
Microscope Gym Python Active, unstable Microscope Gym is a new microscope control software under development.
Zeiss OAD Multiple Active, stable Open Application Development (OAD) for Zen BLUE software to control Zeiss microscopes.
Acquifer API Python Active, stable Acquifier API is a python package providing utilitary functionalities when working with an ACQUIFER Imaging Machine.
smartLLSM Python Active, stable smartLLSM is a pipeline for Smart Lattice Light Sheet Microscopy to image rare and complex cellular events.
Navigate Python Active, stable Navigate is an open source Python package for control of light-sheet microscopes. It allows for easily reconfigurable hardware setups and automated acquisition rotuines.
MicroMator Python Inactive, stable MicroMator is an open and flexible software for reactive microscopy.
Pycromanager Python Active, stable pycromanager is a Python library for customized microscope hardware control and integration with image processing. It works together with Micro-manager and Micro-magellan, or can be used as a standalone backend system for controlling microscope hardware.
pymmcore-plus Python Active, stable pymmcore-plus is sa library to control Micosocpes. It extends pymmcore with a number of features designed to facilitate working with Micro-manager in pure python/C environments.
Python Microscopy Environment (PYME) Python Active, stable PYME is an open-source application suite for light microscopy acquisition, data storage, visualization, and analysis. It offers a particularly comprehensive set of features for single-molecule localization techniques, but also supports many other modalities.
AutoPilot Java Inactive, stable AutoPilot is an open-source project that hosts the general algorithm for the assessment of image quality, an automated computational method for 3D mapping, and data-driven optimization of the system of light-sheet microscopes.
ShrimPy Ptython Active, stable shrimPy is a pythonic framework for high-throughput smart microscopy and high-performance analysis.
Python Microscope Python Active, stable Python's microscope package is a free and open source library for microscope control aimed at those who are building their own microscopes or want programmatic control for microscope experiments.
qi23lab-OPM Python Active, stable qi2lab-OPM I a library for control, reconstruction, and analysis code for multiplexed imaging using oblique plane microscopy (OPM).
exa-spim-control Python Active, stable exa-spim-control is a program for acquisition control for the exaSPIM microscope system.
DiSPIM Python Inactive, stable DiSPIM is a program for the control of the Dual Inverted Selective Plane Illumination Microscopy (DiSPIM) from ASI.
OpenSimView Java Inactive, stable OpenSimView is an open access repository provides all information for building a 2 detection and 2 illumination simultaneous multiview microscope.
mesoSPIM-control Python Active, stable mesoSPIM-control is n image acquisition software for mesoscale selective plane illumination microscope (mesoSPIM) light-sheet microscopes.
leicaautomator Python Inactive, stable Years-old attempt to automate Leica SPX microscopes
leicaimage Python Active, stable Handle Leica Matrix Screener experiment images. The leicaimage library is a modified version of the "leicaexperiment" library.
ExEngine Python Active, stable Toolkit for building microscopy hardware control. ExEngine is a flexible intermediary which enables mixing and matching of components from different frameworks within a single application.
LumaViewPro Python Active, stable Open source control interface for Etaluma microscopes
SpecPy Python Inactive, stable Python utility interface to control iMSPECTOR software, making smart microscopy in Abberior's microscopes possible.
